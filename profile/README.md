## NovaBook 📕 

- MSA 아키텍처로 진행한 온라인 서점 쇼핑몰 프로젝트
<br>
  
### 기능 🎈
-  회원
    - 회원가입, 정보수정, 탈퇴, 마이페이지
- 도서
    - 도서 조회, 검색
- 관리자
    - 도서관리, 회원 관리, 주문 관리
- 주문
   - 도서 주문, 결제, 취소, 환불
- 쿠폰
   - 쿠폰 발급, 사용하여 결제 가능
